# Классификация отзывов через Groq API

## Описание задачи

Скрипт читает отзывы из файла `reviews.csv`, отправляет каждый отзыв в LLM через Groq API и получает структурированный ответ в формате JSON.  
Для каждого отзыва определяется тональность: positive, negative или neutral, а также основная тема отзыва.

## Используемые технологии

- Python
- Groq API
- Модель: llama-3.3-70b-versatile
- CSV для входных данных
- JSON для результата

## Структура проекта

- `main.py` — основной скрипт
- `reviews.csv` — входные данные с отзывами
- `results.json` — результат обработки
- `.env` — файл с API-ключом
- `README.md` — описание проекта

## Установка

```bash
pip install groq python-dotenv
```
